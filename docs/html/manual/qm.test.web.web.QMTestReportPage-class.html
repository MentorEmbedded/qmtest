Class QMTestReportPage

source code

web.DtmlPage --+    
               |    
 DefaultDtmlPage --+
                   |
                  QMTestReportPage

A 'QMTestReportPage' is a 'DtmlPage' for pages generated by QMTest.

A 'QMTestReportPage' automatically looks for DTML templates in the directory that contains QMTest DTML templates.
