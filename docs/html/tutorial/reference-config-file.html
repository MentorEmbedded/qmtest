14. Configuration Variables

These configuration variables are used by QMTest. You should define them in the [qmtest] section of your QM configuration file.

pid-file

The default path to use when creating a PID file with the --pid-file option. (See Section 5, “qmtest gui” for more information about this option.) If this entry is not present, an appropriate platform-specific default value is used.
